## Описание PR
Добавление режима **«Братья по крови» (Blood Brothers)** — парного антагониста для Space Station 14.

**Братья по крови** — это два кандидата на вступление в Синдикат, которые должны работать вместе, чтобы выполнить поставленные цели. Суть и задачи те же, что и у предателей, за одним исключением: **у них нет аплинка**.

Если братья выполнили свои цели и покидают станцию на шаттле эвакуации или спасательных подах, победа достается им. Для этого оба брата должны быть живы и свободны.

## Почему / Баланс
Новый антагонист, ориентированный на командную работу в парах. Братья по крови заполняют нишу между предателями (одиночки) и ядерными оперативниками (команда 4+). Механика парных целей создаёт уникальный геймплей, где братья зависят друг от друга.

**Формирование пар:**
- Пара = строго 2 человека. Если напарника нет — антаг не выдаётся.
- Количество пар рассчитывается от онлайна: `floor(игроки / 10)`, макс. 6 антагов = 3 пары.
- Минимум 20 игроков для запуска режима.

**Цели:**
- Общие цели выдаются всей паре (убийство, кража, эвакуация).
- Удалена старая цель «Убедитесь, что ваш коллега-предатель останется жив».
- Документные цели (опционально, при 2+ парах) — одна из пар получает задание по обмену/захвату/передаче цветных папок.

**Пары между собой:**
- Не могут кооперироваться без общей цели.
- Могут противостоять друг другу.
- В паре нет лидера — братья равноправны.

**Режимы запуска:**
- Secret — 30% вес
- AllAtOnce / AllerAtOnce
- Dynamic — при 15+ игроках

## Техническая информация
- Формирование пар происходит раундстартом в `OnAntagSelect` при накоплении 2+ `PendingMinds`.
- Система сама подбирает пары жадно: при каждом добавлении антага в список проверяется, не набралось ли 2 кандидата.
- Цели выдаются общие для всей пары через `BloodBrotherLinkedObjectivesComponent`.
- Документные цели используют `BloodBrotherDocumentsConditionComponent` с вариантами сценариев (0 — обмен, 1 — захват, 2 — сохранение, 3 — передача, 4 — получение).
- Папки-документы спавнятся в инвентарь одного из братьев через `SpawnItemOnEntity`, описание устанавливается через `MapInitEvent`.
- `BloodBrotherRuleComponent` хранит список команд и ожидающих умов.
- `FilterOutOwnBloodBrotherTeam` исключает членов своей пары из целей на убийство.
